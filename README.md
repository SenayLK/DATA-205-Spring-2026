
**A Statistical/Predictive Analysis of Car Crashes in Montgomery County, MD** 

This is a capstone project for my DATA 205 class at Montgomery College spring 2026.

**Research Goals and Questions**
The primary goal of this project is to statistically analyze collision incidents across Montgomery County, MD to uncover contributing factors and reduce the number of lives being negatively affected by car crashes. Additionally, a secoond goal for this project is to predict and forecast car crashes.


**Data**
The two main datasets I will be using are called Crash Reporting - Incidents and Crash Reporting - Drivers data. Both datasets, which are updated on a weekly basis, are sourced from Data Montgomery, and they were collected via the Automated Crash Reporting System of the Maryland State Police. The Incidents data consists of 120,000+ rows and 37 columns of information and details for each collision incident. Similarly, the Drivers data consists of 212,000+ rows and 39 columns of information and details for each driver involved in a collision. These datasets are based on preliminary information that may change upon further investigation. An issue with these two datasets is that almost all of their columns are categorical so I will have to primarily use count of rows.
